## 2. 复合翼构型

### 2.1 构型特点

复合翼结合了旋翼和固定翼的优势:

```
   升力旋翼(4个, 固定朝上)
   1(左前)    2(右前)
       \   /
   ---- + ----  ← 固定翼 (提供巡航升力)
       /   \
   4(左后)    3(右后)
        \
         → 推进器 (机头方向)
```

### 2.2 与 eVTOL 的区别

| 方面 | eVTOL (倾转旋翼) | 复合翼 (固定旋翼+推进器) |
|------|------------------|------------------------|
| 旋翼 | 可倾转 0°~90° | 固定朝上 |
| 推进 | 旋翼倾转后推进 | 独立推进器 |
| 机械复杂度 | 高 (倾转机构) | 低 (无倾转) |
| 控制复杂度 | 高 (倾转+推力分配) | 中 (独立控制) |
| 效率 | 过渡时效率波动 | 悬停/巡航各自优化 |

## 3. 动力学模型

### 3.1 力的组成

```
F_body = F_lift + F_push + F_aero + gravity
```

- `F_lift`: 4 个升力旋翼推力 (垂直方向)
- `F_push`: 推进器推力 (机头方向)
- `F_aero`: 气动力 (随速度增大)
- `gravity`: 重力

### 3.2 升力旋翼

固定朝上，仅提供垂直力:

```
F_lift_i = k_t * omega_i^2  (沿 body -Z 方向)
```

### 3.3 推进器

沿机头方向:

```
F_push = k_t * omega_push^2  (沿 body +X 方向)
```

### 3.4 气动机翼

随速度增大，机翼提供越来越多的升力:

```
L = 0.5 * rho * V^2 * S * CL(alpha)
D = 0.5 * rho * V^2 * S * CD(alpha)
```

## 4. 飞行模式

### 4.1 悬停模式

- 仅升力旋翼工作
- 推进器关闭
- 适合起降

### 4.2 过渡模式

- 升力旋翼 + 推进器 + 部分气动
- 速度渐增，气动升力渐增
- 升力旋翼负载渐减

### 4.3 巡航模式

- 推进器为主
- 机翼提供主要升力
- 升力旋翼辅助 (低速或机动时)

---

## 5. 能耗特性

### 5.1 悬停

```
P_hover = m*g * v_induced / eta
```

需要持续消耗能量平衡重力。

### 5.2 巡航

```
P_cruise = D * V / eta
```

机翼提供升力，推进器只需克服阻力。

### 5.3 能效对比

| 模式 | 能效 |
|------|------|
| 悬停 | 低 (~200 Wh/km) |
| 巡航 | 高 (~50 Wh/km) |

## 7. 调参

### 升力旋翼

- `lift_kt`: 增大 → 同转速更大推力
- `lift_arm`: 增大 → 更大的力矩 (姿态控制)

### 推进器

- `push_kt`: 增大 → 更大巡航推力

### 气动

- `CL_alpha`: 增大 → 更高效的机翼
- `CD0`: 减小 → 更低阻力
