    ┌───┐      │    ┌─────────────────────────────────┐

    ┌───┐      │    │  [ω₁²]         [T    ]          │
    │τ_θ│──────┼───▶│  [ω₂²] = M⁻¹ · [τ_φ  ]          │
    └───┘      │    │  [ω₃²]         [τ_θ  ]          │
    ┌───┐      │    │  [ω₄²]         [τ_ψ  ]          │

                    │      [0     l·kt  0    -l·kt]   │
                    │      [-l·kt 0     l·kt  0   ]   │
                    │      [kd   -kd    kd   -kd  ]   │
                    └──────────────┬──────────────────┘
                                   │
                    ┌──────────────┴──────────────────┐
                    │         电机限幅                  │
                    │  ω_min² ≤ ωᵢ² ≤ ω_max²          │
                    └──────────────┬──────────────────┘

                    ┌──────────────┴──────────────────┐

                    │      │       │       │      │

                    │      │       │       │
                    ▼      ▼       ▼       ▼

### 3.2 倾转角调度曲线

```
  倾转角 (°)
  90 ┤                              ┌──────────────────────
     │                           ╱
     │                        ╱
     │                     ╱
     │                  ╱
     │               ╱
     │            ╱
     │         ╱
  0  ┤────────╱
     └────┬───────┬───────────┬─────── 空速 (m/s)
          0       5          20
          │       │           │
          │       │           └── 巡航模式阈值
          │       └── 过渡开始
          └── 悬停模式

  调度公式:
    tilt_ratio = (V - V_hover) / (V_cruise - V_hover)
    tilt_angle = tilt_ratio × 90°

  悬停推力权重 = 1 - tilt_ratio
  巡航推力权重 = tilt_ratio
```

## 5. 高级控制策略

### 5.1 LQR 控制器

```
  ┌──────────────────────────────────────────────────────────────────────────────┐
  │                          LQR 姿态控制器                                      │
  └──────────────────────────────────────────────────────────────────────────────┘

    状态: x = [φ, θ, ψ, p, q, r]ᵀ

    ┌──────────┐                         ┌──────────┐
    │ att_des  │                         │ att_cur  │
    │ [φd,θd,ψd]│                        │ [φ,θ,ψ]  │
    └────┬─────┘                         └────┬─────┘
         │                                    │
         ▼                                    ▼
    ┌────┴────┐                          ┌────┴────┐
    │   -     │                          │         │
    └────┬────┘                          │         │
         │                               │         │
       x_err = x_des - x_cur            │         │
         │                               │         │
         ▼                               │         │
    ┌────────────────────────────────────────────────┐
    │                                                │
    │           最优状态反馈                           │
    │                                                │
    │         u = -K · x_err                         │
    │                                                │
    │   ┌──────────────────────────────────────┐     │
    │   │  K = lqr(A, B, Q, R)                │     │
    │   │                                      │     │
    │   │  最小化: J = ∫(xᵀQx + uᵀRu) dt      │     │
    │   │                                      │     │
    │   │  Q = diag(q_φ, q_θ, q_ψ, q_p, q_q, q_r)  │
    │   │  R = diag(r_φ, r_θ, r_ψ)            │     │
    │   │                                      │     │
    │   │  A: 线性化系统矩阵                    │     │
    │   │  B: 输入矩阵                         │     │
    │   └──────────────────────────────────────┘     │
    │                                                │
    └───────────────────────┬────────────────────────┘
                            │
                       u = [τ_φ; τ_θ; τ_ψ]
                            │
                            ▼
                    输出到混控器

  优势: 全局最优, 自然处理多变量耦合
  劣势: 需要精确模型, 线性化点选择
```

### 5.2 反步法 (Backstepping) 控制器

```
  ┌──────────────────────────────────────────────────────────────────────────────┐
  │                       反步法姿态控制器                                        │
  └──────────────────────────────────────────────────────────────────────────────┘

  逐层递推, 每层设计虚拟控制量:

  第一层: 姿态角 → 角速度期望
  ──────────────────────────────
    att_err = att_des - att_cur
         │
         ▼
    ┌─────────────────────────────────────────┐
    │  虚拟控制量 α₁:                          │
    │                                         │
    │  α₁ = -c₁·att_err + att_dot_des        │
    │                                         │
    │  c₁ > 0: 收敛速率                       │
    │  Lyapunov: V₁ = ½·att_errᵀ·att_err     │
    │  V̇₁ = -c₁·‖att_err‖² ≤ 0  (稳定)      │
    └────────────────┬────────────────────────┘
                     │
                ω_des = α₁
                     │
  第二层: 角速度误差 → 力矩
  ──────────────────────────────
                     ▼
    ┌─────────────────────────────────────────┐
    │  角速度误差:                             │
    │  z₂ = ω_cur - ω_des                    │
    │                                         │
    │  力矩命令:                              │
    │  τ = I·(α₂ + att_err + c₁·z₂)         │
    │                                         │
    │  α₂ = -c₂·z₂ + ω_dot_des              │
    │                                         │
    │  c₂ > 0: 角速度收敛速率                 │
    │                                         │
    │  Lyapunov: V₂ = V₁ + ½·z₂ᵀ·I·z₂      │
    │  V̇₂ ≤ -c₁·‖att_err‖² - c₂·‖z₂‖² ≤ 0 │
    └────────────────┬────────────────────────┘
                     │
                τ = [τ_φ; τ_θ; τ_ψ]
                     │
                     ▼
                输出到混控器

  优势: 严格 Lyapunov 稳定性证明, 天然处理非线性
  劣势: 需要角速度导数, 参数调节复杂
```

### 5.3 滑模控制 (SMC) 控制器

```
  ┌──────────────────────────────────────────────────────────────────────────────┐
  │                         滑模姿态控制器                                        │
  └──────────────────────────────────────────────────────────────────────────────┘

    att_err = att_des - att_cur
    att_dot_err = att_dot_des - att_dot_cur

         │
         ▼
    ┌─────────────────────────────────────────────────┐
    │  滑模面设计:                                    │
    │                                                 │
    │  s = att_dot_err + λ · att_err                  │
    │                                                 │
    │  λ > 0: 滑模面斜率, 决定收敛速度               │
    │                                                 │
    │  当 s = 0 时, 系统沿滑模面滑动                  │
    │  → att_dot_err = -λ · att_err                  │
    │  → 指数收敛                                    │
    └────────────────────┬────────────────────────────┘
                         │
                         ▼
    ┌─────────────────────────────────────────────────┐
    │  等效控制 (保证滑模运动):                        │
    │                                                 │
    │  u_eq = I·(att_ddot_des + λ·att_dot_err)       │
    │        + ω × (I·ω)                             │
    │                                                 │
    │  (抵消系统非线性)                               │
    └────────────────────┬────────────────────────────┘
                         │
                         ▼
    ┌─────────────────────────────────────────────────┐
    │  切换控制 (鲁棒性):                              │
    │                                                 │
    │  u_sw = -K · sign(s)                           │
    │                                                 │
    │  或带边界层:                                    │
    │  u_sw = -K · sat(s/ε)                          │
    │                                                 │
    │  ┌─────────────────────────────────┐            │
    │  │  sign(s) =                      │            │
    │  │    +1   if s > 0                │            │
    │  │    -1   if s < 0                │            │
    │  │     0   if s = 0                │            │
    │  │                                 │            │
    │  │  sat(s/ε) =                     │            │
    │  │    s/ε      if |s| ≤ ε          │            │
    │  │    sign(s)  if |s| > ε          │            │
    │  └─────────────────────────────────┘            │
    │                                                 │
    │  ε: 边界层厚度 (减小抖振)                       │
    │  K: 切换增益 (克服不确定性)                     │
    └────────────────────┬────────────────────────────┘
                         │
                         ▼
    ┌─────────────────────────────────────────────────┐
    │  总控制量:                                      │
    │                                                 │
    │  τ = u_eq + u_sw                               │
    │                                                 │
    │  = I·(att_ddot_des + λ·att_dot_err)           │
    │    + ω × (I·ω)                                │
    │    - K · sat(s/ε)                              │
    └────────────────────┬────────────────────────────┘
                         │
                    τ = [τ_φ; τ_θ; τ_ψ]
                         │
                         ▼
                    输出到混控器

  优势: 对模型不确定性和外部扰动鲁棒
  劣势: 抖振 (chattering), 需要边界层平滑
```

### 5.4 控制器对比总结

```
  ┌──────────────────────────────────────────────────────────────────────────────┐
  │                         控制器性能对比                                        │
  └──────────────────────────────────────────────────────────────────────────────┘

  ┌──────────┬──────────┬──────────┬──────────┬──────────┐
  │   指标   │   PID    │   LQR    │ Backstep │   SMC    │
  ├──────────┼──────────┼──────────┼──────────┼──────────┤
  │ 稳定性   │ 局部     │ 全局最优 │ Lyapunov │ 滑模面   │
  │ 鲁棒性   │ 一般     │ 一般     │ 一般     │ 强       │
  │ 调参难度 │ 低       │ 中       │ 高       │ 中       │
  │ 计算量   │ 低       │ 中       │ 中       │ 中       │
  │ 模型依赖 │ 无       │ 需线性化 │ 需模型   │ 需模型   │
  │ 多变量   │ 解耦处理 │ 天然处理 │ 天然处理 │ 天然处理 │
  │ 抗扰动   │ I 项补偿 │ Q/R 调节 │ 一般     │ 强       │
  │ 实现难度 │ ★        │ ★★       │ ★★★      │ ★★       │
  └──────────┴──────────┴──────────┴──────────┴──────────┘

  推荐选择:
  - 快速原型/教学: PID (简单直观)
  - 精确控制/已知模型: LQR (最优性)
  - 非线性/大角度: Backstepping (理论保证)
  - 不确定性/扰动: SMC (鲁棒性)
```
