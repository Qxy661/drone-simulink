- 理解固定翼为什么能飞（升力的物理本质）

**与四旋翼的核心区别**:

| 方面 | 四旋翼 | 固定翼 |
|------|--------|--------|
| 升力来源 | 旋翼推力 | 机翼气动 |
| 能否悬停 | 能 | 不能 |
| 控制方式 | 直接力/力矩 | 舵面偏转+油门 |
| 最低速度 | 0 | >失速速度(~12m/s) |
| 状态维度 | 12 | 12 + 气动状态 |

## 2. 物理直觉 — 用生活类比理解

### 2.1 升力就像把手伸出车窗

开车时把手伸出窗外，手掌微微上倾（迎角 α > 0），气流就会把手往上推。这就是升力：
- **速度越快** → 升力越大（∝ V²）
- **倾斜越大** → 升力越大（∝ CL(α)）
- **倾斜太大** → 气流分离，升力骤降（失速）

### 2.2 阻力就像在水中走路

在水中走路比在空气中难得多，因为水的阻力大。固定翼也一样：
- **速度越快** → 阻力越大（∝ V²）
- **升力越大** → 诱导阻力越大（因为翼尖涡流）

### 2.3 转弯就像骑自行车

骑自行车转弯时，车身倾斜（滚转），重力和升力的合力提供向心力。固定翼转弯完全一样：
- 倾斜滚转角 φ → 升力的水平分量提供转弯力
- 同时需要方向舵配合（协调转弯），否则会侧滑

## 3. 坐标系与状态变量

### 3.1 三个关键角度

```
            来流 V∞
              ↓
              │  α (迎角)
              │ /
    ──────────┼────────── 机体 X 轴
              │
              ↓ 升力 L (垂直于来流)
```

| 角度 | 符号 | 定义 | 物理含义 |
|------|------|------|---------|
| 迎角 | α (alpha) | 机体 X 轴与来流的夹角 | 决定升力大小 |
| 侧滑角 | β (beta) | 来流偏离对称面的角度 | 决定侧力大小 |
| 航向角 | ψ (psi) | 机头在地面的投影方向 | 飞机朝哪飞 |

### 3.2 气动力的三要素

```
升力 L = 0.5·ρ·V²·S·CL(α)     垂直于来流方向
阻力 D = 0.5·ρ·V²·S·CD(α)     平行于来流方向（向后）
侧力 Y = 0.5·ρ·V²·S·CY(β)     侧向（垂直于对称面）
```

**共同因子**: `q̄ = 0.5·ρ·V²` 称为**动压**，单位 Pa。它是气动力的"放大器"。

### 3.3 状态变量 (12 维)

与四旋翼相同：`[x,y,z, vx,vy,vz, φ,θ,ψ, p,q,r]`

但物理解释不同：
- vx,vy,vz 是空速分量（相对于空气），不是地速
- 迎角 α = atan2(vz, vx)，侧滑角 β = asin(vy/V)

## 4. 动力学方程推导

### 4.1 升力系数 CL(α)

**线性区** (α < 失速角):
```
CL(α) = CL₀ + CL_α · α
```
- `CL₀ = 0.2` — 零迎角升力系数（翼型弯度产生）
- `CL_α = 5.0 /rad` — 升力线斜率（理论值 2π，实际受展弦比影响）

**失速后** (α > 15°):
```
CL = CL₀ + CL_α·α_stall - 2·CL_α·(α - α_stall)   (C1 连续失速)
```

**物理含义**: 迎角超过临界值后，气流从机翼上表面分离，升力急剧下降。

```
CL
 │        ╱╲
 │       ╱  ╲  ← 失速
 │      ╱    ╲
 │     ╱      ╲
 │ CL₀╱        ╲
 │───╱──────────╲──→ α
 0       15°
```

### 4.2 阻力系数 CD(α)

```
CD(α) = CD₀ + CL²/(π·e·AR)
```
- `CD₀ = 0.02` — 零升阻力（摩擦+形状阻力）
- `e = 0.8` — Oswald 效率因子
- `AR = b²/S = 8` — 展弦比
- 第二项是**诱导阻力**：升力越大，翼尖涡流越强，阻力越大

### 4.3 气动力矩

| 力矩 | 公式 | 控制面 |
|------|------|--------|
| 俯仰 M | `q̄·S·c·(Cm₀ + Cm_α·α + Cm_q·q·c/2V + Cm_de·δ_e)` | 升降舵 δ_e |
| 滚转 L | `q̄·S·b·(Cl_β·β + Cl_p·p·b/2V + Cl_da·δ_a)` | 副翼 δ_a |
| 偏航 N | `q̄·S·b·(Cn_β·β + Cn_r·r·b/2V + Cn_dr·δ_r)` | 方向舵 δ_r |

**关键导数含义**:
- `Cm_α = -1.0`: 俯仰静稳定（抬头→低头恢复）
- `Cl_p = -0.5`: 滚转阻尼（滚转→减速）
- `Cn_β = 0.06`: 航向静稳定（侧滑→恢复）

### 4.4 舵面效率

| 舵面 | 偏转 | 效率系数 | 力矩效果 |
|------|------|---------|---------|
| 升降舵 δ_e | ±30° | Cm_de = -0.5 | 俯仰力矩 |
| 副翼 δ_a | ±25° | Cl_da = 0.2 | 滚转力矩 |
| 方向舵 δ_r | ±30° | Cn_dr = 0.05 | 偏航力矩 |

### 4.5 六自由度方程

与四旋翼相同的 Newton-Euler 方程，但力的来源不同：

```
平动: m·a = R·(F_aero + F_thrust) + [0,0,-mg]
转动: I·ω̇ = M_aero + M_control - ω×(I·ω)
```

其中气动力在机体坐标下：
```
Fx = -D·cosα + L·sinα + T      (前向: 推力 - 阻力 + 升力分量)
Fy = Y                           (侧向: 侧力)
Fz = -D·sinα - L·cosα           (垂向: 升力主分量)
```

## 5. 控制架构总览

### 5.1 完整控制框图

```
┌──────────┐    ┌──────────────┐
│ Alt_Des  ├───→│              │
│ (100m)   │    │  Alt_Ctrl    ├──→ elevator ──┐
│          │    │  高度控制     ├──→ throttle   │
│ V_Des    ├───→│              │               │
│ (25m/s)  │    └──────────────┘               │
└──────────┘                                   ▼
                                       ┌──────────────┐
┌──────────┐    ┌──────────────┐       │              │
│ Hdg_Des  ├───→│  Hdg_Ctrl    ├──→ aileron ──→│  FW_Dyn     │──→ state(12)
│ (0 rad)  │    │  航向控制     ├──→ rudder  ──→│  6DOF+气动   │
└──────────┘    └──────────────┘       │              │
                                       └──────┬───────┘
                                              │
                         反馈 ←───────────────┘
```

### 5.2 控制链路

```
高度误差 → PID → 俯仰角期望 → PID → 升降舵偏转
空速误差 → PID → 油门开度
航向误差 → PID → 滚转角期望 → PID → 副翼偏转
侧滑角   → P  → 方向舵偏转
```

## 6. Simulink 模型逐块讲解

> 本章对应 `fixedwing_control.slx` 中的每一个子系统和 block

### 6.1 顶层输入

| Block | 类型 | 值 | 作用 |
|-------|------|-----|------|
| `Alt_Des` | Constant | 100 | 期望高度 [m] |
| `Hdg_Des` | Constant | 0 | 期望航向 [rad] |
| `V_Des` | Constant | 25 | 期望空速 [m/s] |

### 6.2 Alt_Ctrl 子系统 (高度控制) — 青色

**输入**: `alt_des`, `V_des`, `fb` [5×1]
**输出**: `elevator`, `throttle`

**反馈信号**: `fb = [z, theta, vx, vy, vz]`

**内部结构**:

```
              ┌─ NegZ(-1) ── Se(+-) ── PID_Alt ── Se2(+-) ── PID_Pitch ── elevator
alt_des ──────┤
fb(1:z) ──────┘

              ┌─ Vx²+Vy²+Vz² → sqrt → V
fb(3:5) ──────┤
              └─ V_Des ── SV(+-) ── PID_V ── throttle
```

**逐块解释**:

| Block | 参数 | 作用 |
|-------|------|------|
| `NegZ` | Gain, -1 | z 取反（NED 下 z 向下为正，高度向上为正） |
| `Se` | Sum, `+-` | 高度误差 = alt_des - (-z) = alt_des + z |
| `PID_Alt` | P=0.5, I=0.02, D=0.3, lim±0.52 | 高度 → 俯仰角期望 (rad) |
| `Se2` | Sum, `+-` | 俯仰角误差 = θ_des - θ |
| `PID_Pitch` | P=2.0, I=0.1, D=0.5, lim±0.52 | 俯仰角 → 升降舵 (rad) |
| `Vx2,Vy2,Vz2` | Math Function, square | 分量平方 |
| `Vsum` | Sum, `+++` | V² = vx²+vy²+vz² |
| `Vsqrt` | Math Function, sqrt | V = √(V²) |
| `SV` | Sum, `+-` | 空速误差 = V_des - V |
| `PID_V` | P=0.3, I=0.05, D=0.01, lim[0,1] | 空速误差 → 油门 (0~1) |

**控制逻辑**:
1. 高度低了 → θ_des 增大（抬头）→ 升降舵上偏 → 机头抬起 → 爬升
2. 空速低了 → 油门增大 → 推力增加 → 加速

### 6.3 Hdg_Ctrl 子系统 (航向控制) — 绿色

**输入**: `hdg_des`, `fb` [5×1]
**输出**: `aileron`, `rudder`

**反馈信号**: `fb = [psi, phi, p, q, r]`

**内部结构**:

```
hdg_des ── Se(+-) ── PID_Hdg ── Se2(+-) ── PID_Roll ── aileron
                ↑                    ↑
fb(1:psi) ──────┘          fb(2:phi) ─┘

fb(3:p) ── Kbeta(-0.5) ── rudder
```

**逐块解释**:

| Block | 参数 | 作用 |
|-------|------|------|
| `Se` | Sum, `+-` | 航向误差 = ψ_des - ψ |
| `PID_Hdg` | P=1.0, I=0.05, D=0.2, lim±0.78 | 航向 → 滚转角期望 (±45°) |
| `Se2` | Sum, `+-` | 滚转角误差 = φ_des - φ |
| `PID_Roll` | P=3.0, I=0.1, D=0.5, lim±0.44 | 滚转角 → 副翼 (±25°) |
| `Kbeta` | Gain, -0.5 | p (滚转角速度) → 方向舵（协调转弯阻尼） |

**控制逻辑**:
1. 航向偏了 → φ_des（倾斜转弯）→ 副翼偏转 → 飞机滚转 → 转弯
2. 滚转角速度 p → 方向舵反向偏转 → 防止侧滑（协调转弯）

### 6.4 FW_Dyn 子系统 (气动动力学) — 红色

**输入**: `elev`, `thr`, `ail`, `rud`
**输出**: `state` [12×1]

**内部结构**:

```
elev ──┐
thr  ──┤
ail  ──┼──→ fw6dof (MATLAB Function) ──→ Integrator ──→ state
rud  ──┤        ↑
Int  ──┘        │
  └─────────────┘
```

| Block | 参数 | 作用 |
|-------|------|------|
| `fw6dof` | MATLAB Function | 6DOF + 气动: state_dot = f(elev,thr,ail,rud,state) |
| `Integrator` | IC=[0;0;-100;25;0;0;0;0.05;0;0;0;0] | 积分得到状态 |

**初始条件含义**:
```
[0; 0; -100;    ← 位置: 北0, 东0, 高度100m (z=-100)
 25; 0; 0;      ← 速度: 北向25m/s巡航, 东0, 垂直0
 0; 0.05; 0;    ← 姿态: 水平, 微仰5°, 航向0
 0; 0; 0]       ← 角速度: 0
```

### 6.5 反馈回路

```
state(12) → Demux(12)
    ├─ 3(z), 8(θ), 4(vx), 5(vy), 6(vz) → Mux_FbAlt(5路) → Alt_Ctrl/3
    └─ 9(ψ), 7(φ), 10(p), 11(q), 12(r) → Mux_FbHdg(5路) → Hdg_Ctrl/3
```

### 6.6 输出

| Block | 作用 |
|-------|------|
| `Scope_Alt` | 双输入: Alt_Des vs z |
| `Scope_Hdg` | 双输入: Hdg_Des vs psi |
| `Scope_Surf` | 舵面偏转 |
| `ToWS` | 记录状态到 `fw_state` |

## 7. MATLAB Function 代码走读

> 对应 `fw6dof` 函数，包含完整气动 6DOF

### 7.1 参数定义

```matlab
m = 10; g = 9.81; rho = 1.225;     % 质量/重力/空气密度
S = 0.5; b = 2.0; c = 0.25;        % 翼面积/翼展/弦长
Ixx = 0.5; Iyy = 1.0; Izz = 1.2; Ixz = 0.02;  % 惯量
```

### 7.2 气动系数

```matlab
CL0 = 0.2; CLa = 5.0; a_stall = 15*pi/180;  % 升力
CD0 = 0.02; e_os = 0.8; AR = b^2/S;           % 阻力
CYb = -0.3;                                    % 侧力
Cm0 = 0.05; Cma = -1.0; Cmq = -15; Cmde = -0.5;  % 俯仰
Clb = -0.05; Clp = -0.5; Clda = 0.2;          % 滚转
Cnb = 0.06; Cnr = -0.1; Cndr = 0.05;          % 偏航
```

### 7.3 空速和气流角

```matlab
V = sqrt(vx^2+vy^2+vz^2) + 1e-6;  % 空速 (加小数防止除零)
alpha = atan2(vz, vx);              % 迎角
beta = asin(vy/V);                  % 侧滑角
```

### 7.4 C1 连续失速模型

```matlab
if alpha < a_stall
    CL = CL0 + CLa*alpha;          % 线性区
else
    CL = CL0 + CLa*a_stall - 2*CLa*(alpha-a_stall);  % 失速后
end
CL = max(min(CL, 1.5), -1.0);      % 限幅
```

### 7.5 气动力计算

```matlab
qbar = 0.5*rho*V^2;    % 动压 [Pa]
L = qbar*S*CL;          % 升力 [N]
D = qbar*S*CD;          % 阻力 [N]
Y = qbar*S*CYb*beta;    % 侧力 [N]
```

### 7.6 舵面力矩

```matlab
M_elev = qbar*S*c*Cmde*elev;    % 升降舵俯仰力矩
L_ail  = qbar*S*b*Clda*ail;     % 副翼滚转力矩
N_rud  = qbar*S*b*Cndr*rud;     % 方向舵偏航力矩
```

### 7.7 气动力矩（含阻尼）

```matlab
M_pitch = qbar*S*c*(Cm0 + Cma*alpha + Cmq*q*c/(2*V)) + M_elev;
M_roll  = qbar*S*b*(Clb*beta + Clp*p*b/(2*V)) + L_ail;
M_yaw   = qbar*S*b*(Cnb*beta + Cnr*r*b/(2*V)) + N_rud;
```

**阻尼项**: `Cmq*q*c/(2*V)` — 俯仰角速度越大，阻尼力矩越大（自动减速）。

### 7.8 力的转换

```matlab
ca=cos(alpha); sa=sin(alpha);
Fx = -D*ca + L*sa + T;    % 前向: 推力 + 升力分量 - 阻力分量
Fz = -D*sa - L*ca;         % 垂向: 升力主分量
Fy = Y;                     % 侧向: 侧力
```

## 11. 故障排查 FAQ

### Q1: 飞机掉高度

**原因**: 空速不足或升力不够

**解决**: 增大 V_Des 或检查 CL 参数。

### Q2: 飞机不断爬升

**原因**: 高度 PID 积分饱和

**解决**: 减小 Ki_alt 或增加积分限幅。

### Q3: 转弯时掉高度

**原因**: 滚转后升力的垂直分量减小

**解决**: 增大 Kp_alt 或在转弯时补偿油门。

### Q4: 舵面剧烈振荡

**原因**: PID 增益过大

**解决**: 降低 Kp_pitch 和 Kp_roll。

### Q5: 飞机不转弯

**原因**: 航向误差被限制在 ±π 内

**解决**: 检查航向角是否跨越 ±π 边界。

### Q6: 仿真发散

**原因**: 步长太大或初始条件不合理

**解决**: 减小 FixedStep (如 0.001)，检查初始空速 > 失速速度。
