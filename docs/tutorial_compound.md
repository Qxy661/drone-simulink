## 2. 物理直觉

### 2.1 复合翼 = 直升机 + 飞机

```
eVTOL (倾转旋翼):              复合翼 (固定旋翼):
   旋翼可以转                    旋翼固定朝上
   一套执行器做两件事              三套执行器各司其职
   机械复杂                      机械简单

   ↑↑↑↑   ↗↗↗↗   →→→→          ↑↑↑↑   ↑↑↑↑   ↑↑↑↑
   悬停    过渡    巡航           悬停    过渡    巡航
                                 +推进器  +推进器 +推进器
```

### 2.2 三套执行器

| 执行器 | 数量 | 功能 | 控制对象 |
|--------|------|------|---------|
| 升力旋翼 | 4 | 垂直升降 | 高度 (悬停/过渡) |
| 推进器 | 1 | 前向推力 | 空速 (巡航) |
| 舵面 | 3 | 气动控制 | 姿态 (巡航) |

### 2.3 为什么选择复合翼？

- **优点**: 机械简单（旋翼不转）、可靠性高、维护方便
- **缺点**: 悬停时旋翼+机翼都是" dead weight"，效率低于纯多旋翼
- **应用**: 城市空中交通 (UAM)、物流无人机

## 3. 坐标系与状态变量

12 维状态：`[x,y,z, vx,vy,vz, φ,θ,ψ, p,q,r]`

**额外输入** (5 个控制量):
- `lift_cmd` — 升力旋翼转速 (rad/s)
- `push_cmd` — 推进器转速 (rad/s)
- `elevator` — 升降舵偏转 (rad)
- `aileron` — 副翼偏转 (rad)
- `rudder` — 方向舵偏转 (rad)

## 4. 动力学方程推导

### 4.1 升力旋翼 (固定朝上)

```
T_lift = 4 · kt · ω_lift²
```
- 4 个旋翼，X 型布局，固定朝上
- 产生纯垂向推力
- 通过差动产生滚转/俯仰/偏航力矩

### 4.2 推进器 (机头方向)

```
T_push = kt_push · ω_push²
```
- 1 个推进器，固定朝机头方向
- 产生纯前向推力
- `kt_push = 3.0e-5` (比升力旋翼大，因为螺旋桨更大)

### 4.3 气动力 (巡航时有效)

```
L = 0.5·ρ·V²·S·CL(α)     机翼升力
D = 0.5·ρ·V²·S·CD(α)     阻力
```

与固定翼相同的气动模型，含 C1 连续失速。

### 4.4 合力

```
Fx = T_push - D·cosα + L·sinα     前向
Fy = Y                              侧向
Fz = T_lift - D·sinα - L·cosα      垂向
```

### 4.5 模式特点

| 模式 | 升力来源 | 推力来源 | 姿态控制 |
|------|---------|---------|---------|
| 悬停 | 升力旋翼 100% | 推进器 0% | 旋翼差动 |
| 过渡 | 升力旋翼 部分 | 推进器 部分 | 旋翼+舵面 |
| 巡航 | 机翼升力 100% | 推进器 100% | 舵面 |

## 6. Simulink 模型逐块讲解

### 6.1 顶层输入

| Block | 类型 | 值 | 作用 |
|-------|------|-----|------|
| `Alt_Cmd` | Step | 0→100 @ t=0 | 期望高度 |
| `V_Cmd` | Step | 0→25 @ t=20 | 期望速度（20秒后加速） |

### 6.2 Mode_Mgr 子系统 (模式管理器) — 青色

**输入**: `alt_cmd`, `V_cmd`, `fb` [6×1]
**输出**: `mode`, `V_des`

**内部结构**:

```
V_cmd → Vsum → Vsqrt → DZ → Sat → mode
                                   → V_des
```

| Block | 参数 | 作用 |
|-------|------|------|
| `Vsum` | Sum of Elements | 速度分量求和 |
| `Vsqrt` | sqrt | 空速大小 |
| `DZ` | Dead Zone, [-5,-5] | 死区（低速不切换模式） |
| `Sat` | Saturation, [0,2] | mode ∈ {0, 1, 2} |

**模式判断**:
- V < 5 m/s → mode = 0 (悬停)
- 5 < V < 20 m/s → mode = 1 (过渡)
- V > 20 m/s → mode = 2 (巡航)

### 6.3 Lift_Ctrl 子系统 (升力旋翼控制) — 绿色

**输入**: `mode`, `alt_des`
**输出**: `lift_cmd`

**内部结构**:

```
alt_des → Se(+-) → PID_Z → Sw(++) → lift_cmd
                       ↑
w0(532) ───────────────┘
```

| Block | 参数 | 作用 |
|-------|------|------|
| `w0` | Constant, 532 | 悬停转速 = sqrt(15×9.81/(4×1.3e-5)) |
| `PID_Z` | P=2.0, I=0.3, D=1.0, lim±300 | 高度 PID |

### 6.4 Push_Ctrl 子系统 (推进器控制) — 黄色

**输入**: `V_des`
**输出**: `push_cmd`

**内部结构**:

```
V_des → G(×60) → Sat([0,2000]) → push_cmd
```

| Block | 参数 | 作用 |
|-------|------|------|
| `G` | Gain, 60 | 速度→转速 (25m/s → 1500 rpm) |
| `Sat` | Saturation, [0,2000] | 转速限幅 |

### 6.5 Surf_Ctrl 子系统 (舵面控制) — 紫色

**输入**: `mode`
**输出**: `elev`, `ail`, `rud`

**简化实现**: 巡航模式前舵面为 0（姿态控制由旋翼负责）

```matlab
elevator = 0;   % 后续可接入姿态 PID
aileron = 0;
rudder = 0;
```

### 6.6 CW_Dyn 子系统 (复合翼动力学) — 红色

**输入**: `lift`, `push`, `elev`, `ail`, `rud`
**输出**: `state` [12×1]

**内部结构**:

```
lift →┐
push →┤
elev →┼──→ cw6dof (MATLAB Function) ──→ Int ──→ state
ail  →┤        ↑
rud  →┘        │
Int ───────────┘
```

| Block | 参数 | 作用 |
|-------|------|------|
| `cw6dof` | MATLAB Function | 全包线 6DOF |
| `Integrator` | IC=[0;0;-50;...] | 初始高度 50m |

## 7. MATLAB Function 代码走读

### 7.1 参数

```matlab
m = 15; g = 9.81; rho = 1.225;
S = 1.0; b = 3.5; c = S/b; AR = b^2/S;
Ixx = 1.0; Iyy = 2.0; Izz = 2.5;
kt = 1.3e-5; kd = 1.8e-7; arm = 1.0;
```

### 7.2 升力旋翼推力

```matlab
T_lift = 4*kt*lift^2;      % 4个旋翼总推力
```

### 7.3 推进器推力

```matlab
T_push = 3.0e-5*push^2;    % 推进器推力（更大系数）
```

### 7.4 气动力 (C1 连续失速)

```matlab
if alpha < a_stall
    CL = CL0 + CLa*alpha;
else
    CL = CL0 + CLa*a_stall - 2*CLa*(alpha-a_stall);
end
CD = CD0 + CL^2/(pi*e_os*AR);
```

### 7.5 合力

```matlab
ca=cos(alpha); sa=sin(alpha);
Fx = T_push - D*ca + L*sa;    % 前向
Fy = Y;                         % 侧向
Fz = T_lift - D*sa - L*ca;     % 垂向
```

### 7.6 旋翼力矩

```matlab
tau_roll  = arm*kt*lift^2 * 0.5;   % 滚转力矩
tau_pitch = arm*kt*lift^2 * 0.3;   % 俯仰力矩
tau_yaw   = kd*lift^2 * 2;          % 偏航力矩
```

### 7.7 总力矩

```matlab
p_dot = (M_roll+tau_roll + (Iyy-Izz)*q*r) / Ixx;
q_dot = (M_pitch+tau_pitch + (Izz-Ixx)*p*r) / Iyy;
r_dot = (M_yaw+tau_yaw + (Ixx-Iyy)*p*q) / Izz;
```

## 11. 故障排查 FAQ

### Q1: 过渡时高度大幅下降

**原因**: 升力旋翼减小太快，机翼升力不足

**解决**: 增大 w0 或降低 V_Cmd 增长速率。

### Q2: 巡航时速度不增加

**原因**: 推进器推力不足或阻力太大

**解决**: 增大 push_kt 或降低 CD0。

### Q3: 舵面无效

**原因**: Surf_Ctrl 输出为 0（简化实现）

**解决**: 接入姿态 PID 控制器。

### Q4: 仿真发散

**原因**: 初始条件不合理

**解决**: 检查 Integrator 初始值，确保初始高度 > 0。
