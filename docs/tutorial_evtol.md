## 2. eVTOL 构型

### 2.1 倾转旋翼构型

本项目采用 4 倾转旋翼构型:

```
   1(左前)    2(右前)
       \   /
   ---- + ----
       /   \
   4(左后)    3(右后)
```

每个旋翼可独立倾转:
- **0° (悬停)**: 推力沿 body Z 轴 (向下)
- **90° (巡航)**: 推力沿 body X 轴 (向前)

### 2.2 飞行模式

| 模式 | 倾转角 | 特点 |
|------|--------|------|
| 悬停 | 0° | 类四旋翼, 垂直升降 |
| 过渡 | 0°~90° | 混合控制, 速度渐变 |
| 巡航 | 90° | 类固定翼, 高效飞行 |

---

## 3. 倾转机构

### 3.1 倾转角定义

```
tilt = 0°   → 悬停 (推力向下)
tilt = 45°  → 过渡 (推力斜向下前)
tilt = 90°  → 巡航 (推力向前)
```

### 3.2 倾转动力学

一阶惯性响应:

```
tilt_dot = (1/tau) * (tilt_cmd - tilt)
```

- `tau`: 倾转机构时间常数 (~0.5s)
- 限速: 15°/s (安全约束)

### 3.3 推力方向

在机体系中:

```
F_dir = [sin(tilt); 0; -cos(tilt)]
```

- tilt=0: F = [0; 0; -1] (向下)
- tilt=90°: F = [1; 0; 0] (向前)

---

## 4. 过渡控制

### 4.1 速度调度

倾转角基于空速自动调度:

```
V < V_hover     → 悬停模式 (tilt=0°)
V_hover < V < V_cruise → 过渡模式 (tilt 渐增)
V > V_cruise    → 巡航模式 (tilt=90°)
```

### 4.2 控制策略

**悬停模式** (类四旋翼):
```
位置环 → 姿态期望 → 姿态环 → 力矩 → 混控 → 电机
```

**过渡模式** (混合):
```
位置环 → 悬停推力 (权重 1-α)
高度环 → 巡航推力 (权重 α)
混合 → 电机
```

**巡航模式** (类固定翼):
```
高度环 → 升降舵 + 油门
航向环 → 副翼 + 方向舵
```

## 5. 能耗模型

### 5.1 功率计算

```
P = Σ(k_t * ω³) / (η_motor * η_prop)
```

- `k_t`: 推力系数
- `ω`: 电机转速
- `η_motor`: 电机效率 (~90%)
- `η_prop`: 螺旋桨效率 (~85%)

### 5.2 能耗对比

| 模式 | 功率 | 能效 |
|------|------|------|
| 悬停 | 高 | 低 (需持续推力平衡重力) |
| 巡航 | 低 | 高 (机翼提供升力) |

## 7. 调参指南

### 7.1 倾转机构

- `tau_tilt`: 增大 → 倾转更慢 (更安全)
- `tilt_rate`: 减小 → 限速更严

### 7.2 悬停控制

与四旋翼调参相同，参见 [tutorial_quad.md](tutorial_quad.md)。

### 7.3 巡航控制

与固定翼调参相同，参见 [tutorial_fixedwing.md](tutorial_fixedwing.md)。

### 7.4 过渡控制

- `hover_speed`: 悬停模式上限 (5 m/s)
- `cruise_speed`: 巡航模式下限 (20 m/s)
- 过渡区间越大越平滑，但效率降低

---

## 8. 常见问题

### Q: 过渡时高度掉?

- 增大过渡区间的悬停推力权重
- 减慢倾转速率

### Q: 巡航时速度不稳?

- 调整油门 PID
- 检查气动模型参数

### Q: 能耗过高?

- 尽快进入巡航模式 (机翼提供升力)
- 优化倾转角调度
