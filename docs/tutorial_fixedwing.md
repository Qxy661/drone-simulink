## 2. 固定翼 vs 四旋翼

| 方面 | 四旋翼 | 固定翼 |
|------|--------|--------|
| 升力来源 | 旋翼推力 | 机翼气动升力 |
| 悬停 | 可以 | 不可以 (需最低速度) |
| 控制输入 | 4 个电机转速 | 升降舵+副翼+方向舵+油门 |
| 动力学 | Newton-Euler | Newton-Euler + 气动模型 |
| 状态 | 12 维 | 12 维 (相同) |

## 3. 气动基础

### 3.1 气流角

```
迎角 (alpha): 速度矢量与机头水平面的夹角
  alpha = atan2(w, u)

侧滑角 (beta): 速度矢量偏离对称面的角度
  beta = asin(v / V)

空速 (V): 速度矢量大小
  V = sqrt(u² + v² + w²)
```

### 3.2 气动力

在气流坐标系中定义:

```
升力 L: 垂直于来流 (向上)
阻力 D: 平行于来流 (向后)
侧力 Y: 垂直于 L 和 D
```

### 3.3 气动系数

```
升力: L = 0.5 * rho * V² * S * CL(alpha)
阻力: D = 0.5 * rho * V² * S * CD(alpha)
侧力: Y = 0.5 * rho * V² * S * CY(beta)
```

其中:
- `rho`: 空气密度 [kg/m³]
- `V`: 空速 [m/s]
- `S`: 翼面积 [m²]
- `CL`, `CD`, `CY`: 无量纲气动系数

### 3.4 升力系数曲线

```
CL(alpha) = CL0 + CL_alpha * alpha     (线性区)
```

- `CL0`: 零迎角升力系数 (~0.2)
- `CL_alpha`: 升力线斜率 (~5.0 /rad)

**失速**: 当 alpha 超过临界值 (~15°), CL 急剧下降。

## 4. 配平

配平是找到使飞机定常飞行的控制输入。

### 4.1 平飞配平条件

```
升力 = 重力:    L = m * g
推力 = 阻力:    T = D
俯仰力矩 = 0:  Cm = 0
```

### 4.2 配平计算

```matlab
aero_model = AeroModel(aero, fw);
[alpha_trim, de_trim, T_trim] = aero_model.trim(fw.V_cruise, 0);
```

输出:
- `alpha_trim`: 配平迎角 (需产生足够升力)
- `de_trim`: 配平升降舵 (消除俯仰力矩)
- `T_trim`: 配平推力 (平衡阻力)

### 4.3 验证配平

```matlab
test_trim
```

## 5. 控制架构

### 5.1 高度控制

```
期望高度 ──→ [高度 PID] ──→ 期望俯仰角 ──→ [俯仰 PID] ──→ 升降舵
期望空速 ──→ [油门 PID] ──→ 油门
```

### 5.2 航向控制

```
期望航向 ──→ [航向 PID] ──→ 期望滚转角 ──→ [滚转 PID] ──→ 副翼
侧滑角   ──→ [偏航 PID] ──→ 方向舵 (协调转弯)
```

### 5.3 协调转弯

转弯时需要滚转产生向心力:

```
tan(phi) = V² / (g * R)
```

- `phi`: 滚转角
- `V`: 空速
- `R`: 转弯半径

方向舵用于消除侧滑，保持协调。

## 8. 调参指南

### 8.1 高度控制

1. 先调俯仰环 (内环):
   - `Kp_pitch`: 增大 → 响应更快, 但可能振荡
   - `Kd_pitch`: 增大 → 抑制振荡
   - `Ki_pitch`: 增大 → 消除稳态误差

2. 再调高度环 (外环):
   - `Kp_alt`: 增大 → 高度跟踪更快
   - `Kd_alt`: 增大 → 抑制高度振荡

3. 油门环:
   - `Kp_throttle`: 增大 → 速度响应更快

### 8.2 航向控制

1. 先调滚转环 (内环):
   - `Kp_roll`: 增大 → 滚转响应更快
   - `Kd_roll`: 增大 → 抑制滚转振荡

2. 再调航向环 (外环):
   - `Kp_heading`: 增大 → 航向跟踪更快

3. 方向舵:
   - `Kp_yaw`: 增大 → 协调性更好

## 9. 常见问题

### Q: 高度持续振荡?

- 减小 `Kp_alt` 或增大 `Kd_alt`
- 检查俯仰环是否稳定

### Q: 转弯时掉高度?

- 增大 `Kp_alt`
- 检查油门是否足够 (转弯阻力增大)

### Q: 侧滑角大?

- 增大 `Kp_yaw` (方向舵增益)
- 检查副翼-方向舵耦合

### Q: 速度不稳定?

- 调整油门 PID
- 检查阻力模型是否合理
